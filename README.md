# Kambaz Quizzes – Frontend

A Next.js 15 + Redux Toolkit implementation of the Kambaz quizzes experience.
The deployed landing page exposes navigation to the LMS itself as well as the
required project overview and team information.

## Getting Started (local)

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` to view the landing page. Use the
“Launch Kambaz” button to access the LMS experience or the “Meet the Team”
button to review the deliverable details.
